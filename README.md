# Blog Web Application - Frontend

A simple frontend web application for a blogging platform where users can read and comment on articles, and authors can create and manage their own posts.

---

## Features

### User Features
- View all published articles
- Read full article details
- Comment on articles
- View author profiles
- User profile page showing all published articles

### Author Features
- Create new articles
- Edit or delete own articles
- View all authored articles
- Manage profile information

---

## Pages

### Home Page
- Displays all published blog articles
- Search and filter articles
- Pagination support

### Article Details Page
- Full article content
- Comments section
- Add new comment

### Login / Register Page
- User authentication
- Author login support

### Dashboard (Author)
- List of authored articles
- Create new article button
- Edit/Delete article actions

### Create Article Page
- Article title
- Article content editor
- Publish button

### User Profile Page
- User information
- List of published articles by the user

---

## Tech Stack

- React.js
- React Router
- Axios 
- Tailwind CSS 
- Context API / Redux
